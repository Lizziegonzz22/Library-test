# Library-test
El proyecto consiste en desarrollar un sistema de gestión de biblioteca en C++ que permita a los usuarios realizar las siguientes opciones: 

- Agregar libros a la biblioetca con título y autor 
- Ver la lista completa de libros en la biblioteca
- Buscar libros por nombre
- Eliminar libros de la biblioteca
- Salir del programa

El sistema debe ser interactivo y presentar un menú al usuario para que seleccione la opción deseada. El programa debe ser capaz de manejar una lista de libros de tamaño variable y realizar las operaciones de manera eficiente.
